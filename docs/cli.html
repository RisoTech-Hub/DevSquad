CLI Commands Reference

General

Command Description
devsquad config Interactive configuration setup. Prompts for slack_app_token and slack_bot_token. Non-interactive mode accepts flags.

Daemon Management

Command Description
devsquad daemon start Kills stale listener processes, loads LaunchAgent com.devsquad.listener, and starts project message processors.
devsquad daemon stop Unloads listener LaunchAgent and all registered project processor LaunchAgents.
devsquad daemon restart Performs a stop followed by a start sequence.
devsquad daemon remove Removes LaunchAgent plists from disk.
devsquad daemon status Queries launchctl list for listener/processors. Reports PID and exit codes.
devsquad daemon logs [--project <name>] Tails listener log. Provide --project to tail a specific processor log.

Project Management

Command Description
devsquad project init Creates Slack channel, invites core members, launches tmux session with Claude CLI, registers project, and starts processor daemon.
devsquad project list Lists all registered projects with their channel and tmux details.
devsquad project update Updates project phase/task in the Slack status message. Used primarily by AI orchestrators.
devsquad project agent Updates a specific agent's status in the project status message.
devsquad project stop Gracefully terminates tmux session (q\n), marks status Offline, and stops the processor daemon.
devsquad project resume Restarts tmux session with --resume, restarts processor daemon, updates status.
devsquad project add Manually registers an existing channel + tmux session as a project.
devsquad project remove Removes project, archives channel, kills tmux session, removes processor daemon.
devsquad project config Shows current configuration for a given project.

Slack Operations

Command Description
devsquad slack send "<message>" Posts a message to the project's channel. Returns message timestamp.
devsquad slack reply <ts> "<message>" Posts a threaded reply to an existing message using its timestamp (ts).

Internal Commands

These commands are spawned automatically by LaunchAgents and should not be run directly.

Command Description
devsquad _run-listener Executes SlackListenerDaemon to fetch from WebSocket.
devsquad _run-processor <project> Executes MessageProcessorDaemon for a specific project.

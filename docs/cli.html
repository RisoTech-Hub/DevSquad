CLI Reference

The devsquad CLI exposes commands for managing every aspect of your agentic workflows.

Global Options

Option Description
-V, --version Output the version number
-h, --help Display help for command

Configuration config

View or set devsquad configurations like Slack tokens.

Command Description
devsquad config Interactive configuration setup. Prompts for Slack tokens.
devsquad config --bot-token <token> Set Slack Bot Token (xoxb-...)
devsquad config --app-token <token> Set Slack App Token (xoxa-...)
devsquad config --view View current configuration
devsquad config --set <key=value> Set a config key (e.g. --set protocol_base=/path)

Project Management project

Manage the lifecycle of a Slack channel ↔ tmux session mapping.

Command Description
devsquad project init Create Slack channel, start tmux, install daemon. Flags: --name, --users, --repo, --local-only
devsquad project stop Gracefully exit Claude session and kill tmux
devsquad project resume Restart tmux session and processor daemon
devsquad project list List all registered projects
devsquad project update Internal command used by orchestrator to update status/phase
devsquad project agent Update agent status in project status message
devsquad project add Manually register a project
devsquad project remove Remove a project and clean up resources
devsquad project config Show config and state for a project
devsquad project set-mode Set project gate mode (autonomous or supervised)

Background Daemons daemon

Manage the LaunchAgents that bridge Slack to local processes.

Command Description
devsquad daemon start Start listener and processors
devsquad daemon stop Stop listener and processors
devsquad daemon restart Restart listener and processors
devsquad daemon remove Remove daemon plists
devsquad daemon status Show PID and load status
devsquad daemon logs Tail daemon logs. Flag: --name <project>

Agent Registry agent

Manage the agents.json registry of your AI team.

Command Description
devsquad agent list List all registered agents
devsquad agent add Register a new agent. Flags: --name, --role, --model, --container, --skills
devsquad agent show <name> Show full details for an agent
devsquad agent remove <name> Remove an agent from the registry
devsquad agent init Seed agents.json with default team agents

Slack Operations slack

Send messages to Slack channels.

Command Description
devsquad slack send <message> Send a message to a channel
devsquad slack reply <threadTs> <message> Reply to a thread
devsquad slack react <ts> <emoji> Add a reaction to a message
devsquad slack unreact <ts> <emoji> Remove a reaction from a message
devsquad slack upload <filepath> Upload a file to a channel

GitHub Integration task

Command Description
devsquad task brief --role <role> Scaffold a pre-filled task brief skeleton. Flags: --role, --task, --output, --title, --local-only

System Health doctor

Command Description
devsquad doctor Run diagnostics on configuration, tokens, daemons, and tmux
devsquad doctor --fix Automatically repair fixable issues

Internal Commands

Warning

These commands are spawned automatically by LaunchAgents and should not be run directly.

Command Description
devsquad _run-listener Internal: run the Slack listener process (used by LaunchAgent)
devsquad _run-processor <project> Internal: run the message processor for a project (used by LaunchAgent)

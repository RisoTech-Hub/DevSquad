devsquad

A CLI tool bridging Slack with Claude Orchestrator sessions.

TypeScript Node.js Slack Socket Mode Redis tmux

devsquad enables development teams to interact seamlessly with AI agents through Slack messages. By listening to Slack via Socket Mode, devsquad reliably routes messages through Redis queues and injects them directly into local tmux sessions running the Claude CLI.

Quick Summary

  • Seamless AI Interaction: Chat with AI orchestrator agents directly from a Slack project channel.
  • Resilient Message Queuing: Uses Redis to queue messages between Slack listeners and tmux processors.
  • State Persistence: Capable of resuming sessions, crashing/restarting daemons without losing queued commands.
  • Robust Daemon Management: Integrates with macOS LaunchAgents to persist background listeners and processor daemons.
  • Status Tracking: Real-time status reporting for individual projects and the AI team container infrastructure.

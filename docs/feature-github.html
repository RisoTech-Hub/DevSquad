GitHub Integrations

devsquad integrates with GitHub to sync tasks and project state. This includes creating issues, linking to GitHub Projects, and synchronizing status updates.

Task Briefs

Use devsquad task brief to scaffold pre-filled task briefs based on agent roles. This generates structured Markdown briefs that guide agents on their specific tasks.

Creating a Task Brief

# Generate a task brief for a Developer role
devsquad task brief --role developer --task "Implement user authentication"

# Save to a file
devsquad task brief --role developer --task "Implement user authentication" --output task.md

# Create with GitHub Issue (requires gh CLI authentication)
devsquad task brief --role developer --task "Implement user authentication" --title "User Authentication Feature"

GitHub Issue Creation

When authenticated via the gh CLI, the task brief command can automatically:

  1. Create a GitHub Issue with the task brief as the body
  2. Link the issue to your repository's Project board
Warning

GitHub Issue creation is fail-open — if it fails (e.g., no auth, network error), the task brief is still generated but the issue is skipped without blocking the workflow.

Project Initialization with GitHub

When running devsquad project init, you can link to GitHub:

# With GitHub repo
devsquad project init --name my-feature --repo owner/repo

# Skip GitHub integration
devsquad project init --name my-feature --local-only

GitHub Projects Integration

During project initialization, if authenticated with gh:

  1. The CLI detects the repository from git remote
  2. Prompts to select a GitHub Project (or skips if none)
  3. Links the project for status synchronization

Status Synchronization

When agents update their status (via devsquad project agent), the status can sync to:

  • Slack status messages in the project channel
  • GitHub Project board items (when linked)

Authentication

Ensure your gh CLI is authenticated before using GitHub features:

# Check authentication status
gh auth status

# Authenticate if needed
gh auth login
